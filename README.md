## Hotwired cases

A collection of cases where you can apply Hotwire in your Rails applications.

### Setup

__Requirements__

- Ruby 3.1.2
- Postgresql
- Foreman

__Run app__

Setup with 
```
bin/setup
```

Seed database with

```
rails db:seed 
or 
rails db:seed:replant
```

Start the server and build assets

```
bin/dev
```
